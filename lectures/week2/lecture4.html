Tidy data

  • The wider data frame is somehow different from other data tables that we have looked at. (Why?)
  • Data frames that have one row per observation and one column per variable are called tidy data.
  • The original data frame is tidy, but the pivoted data frame is not.
  • tidyverse libraries expect tidy data as input, and generally return tidy data as output. (Whence the name.)
